# BookEase Blockchain

A decentralized booking and escrow system with trust scoring built on Ethereum.

## Vision

To remove fraud and trust issues in online bookings using smart contracts.

## Features (Planned)

- Blockchain-based escrow payments
- Automatic refund system
- Trust scoring for providers
- Transparent booking history

## Tech Stack

- Solidity
- Hardhat
- React
- Ethers.js
